翻转字符串

题目：

先把字符串转化为数组，再借助数组的reverse方法翻转数组顺序，最后把数组转化成字符串。

代码如下：

        function reverseString(str){
          console.log(str);     //这行代码用于调试结果
          return str;
        }

        reverseString("hello");

要求：

  1. reverseString("hello") 应该返回一个字符串
  2. reverseString("hello") 应该返回"olleh"
  3. reverseString("Howdy") 应该返回"ydwoH"
  4. reverseString("Greetings from Earth") 应该返回"htraE morf sgniteerG"
解法：
        function reverseString(str){
          console.log(str.split('').reverse().join(""));
        }

        reverseString("hello");
