今天在查询问题的时候，看到这种写法，一下没反应过来，代码如下：

<script>
var o = {
      valueOf: function(){
        return -1;
      }
    };
o = +o;
console.log(o);  //-1
</script>

这里面涉及到valueOf()方法，后面我也会对这个做详细的整理；简单的说，就是valueOf()方法会在后台进行隐式调用，在运行 o = +o 的时候，等号右边已经进行valueOf()方法的调用，也就是 o = +(-1);
